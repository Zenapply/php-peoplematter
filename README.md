## Installation

Install via [composer](https://getcomposer.org/) - In the terminal:
```bash
composer require zenapply/php-peoplematter
```

## Usage

```php
use Zenapply\PeopleMatter\Api\PeopleMatter;
$client = new PeopleMatter("api_key");
```
